External CSS

Style lives outside the HTML

All the design on this page comes from a separate styles.css file — linked with a single line in the <head>.

When CSS lives in its own file, one stylesheet can control the look of every page on a website. Change a color once — it updates everywhere instantly.

External CSS is the standard way to style real websites. It keeps structure (HTML) and presentation (CSS) cleanly separated, making both easier to read, edit, and maintain.
