Internal CSS · <style> in <head>

Background in warm cream

Internal CSS is written inside a <style> tag placed in the <head> of the HTML document. The warm cream background on this page is set entirely through internal CSS.

<style> body { background-color: #fdf6ec; font-family: 'Jost', sans-serif; min-height: 100vh; } </style>

No external file needed — the styles travel with the page, making it self-contained and easy to share as a single document.
